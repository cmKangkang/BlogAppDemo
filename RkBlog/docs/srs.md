# 软件需求

## 模块划分

1. 用户
   1. 用户名：id
   2. 密码：pwd
   3. 昵称：nickname
   4. 头像：avatar
2. 文章
   1. 标题：title
   2. 内容：content
   3. banner图，可选：banner
   4. 发布时间：release
   5. 作者：author
   6. 标签(多个)：tag[]
   7. 浏览量：pageviews

## 页面

1. 首页
   1. 文章列表：按照时间、浏览量排序
2. 搜素页
   1. 关键词搜索
   2. 标签搜索
3. 文章详情
   1. md文章内容解析展示
   2. 代码高亮：highlight.js
4. 登录
   1. 用户名与密码登录
5. 注册
6. 个人中心
   1. 展示自己的文章列表
   2. 新建、发布文章
   3. 编辑文章
   4. 删除文章
   5. 修改密码与头像
